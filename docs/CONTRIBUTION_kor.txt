# 컨트리뷰션

OpenCP 플랫폼에 기여하기로 선택해 주셔서 감사합니다. 세상에는 수많은 훌륭한 오픈 소스 프로젝트가 있으므로 OpenCP 플랫폼에 기여하는 데 관심을 가져주셔서 감사합니다. 

본 오픈 소스 프로젝트는 누구나 자유롭게 다양한 컨트리뷰션에 참가하여 활동 할 수 있습니다.
참여 가능한 대상은 본 프로젝트를 사용하는 사용자부터, 본 프로젝트에 컨트리뷰션 활동을 하는 모든 사람입니다. 소스 수정에 대한 커미터와 메인테이너는 본 공개소프트웨어를 제공하는 한국전자통신연구원의 오픈소스센터가 됩니다.

참여영역
1.	소스 코드 수정
2.	오타 수정
3.	번역
4.	가이드 문서 작성
5.	디자인 수정
6.	의견 제시
